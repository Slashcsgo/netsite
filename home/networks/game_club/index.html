Game club

This is the common solution for game club network.

You are also able to download .pkt file to see details of that network.

Without going into details, we can simply say that the network consists of three subnets:

  • Network for local administrators - 192.168.0.0(/16)
  • Network for guests (DHCP is used)
  • Gaming network itself (10.0.0.0(/8)). This one can be extended or splitted more (scalable). If you ran out of NIs on switch you can add another one using GigabitEthernet0/1 as incoming and GigabitEthernet0/2 as outgoing in a list.
